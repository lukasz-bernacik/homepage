HTML - HyperText Markup Language

Język HTML to język znaczników. Odpowiada on za właściwe opakowanie treści. To główny szkielet strony internetowej. HTML pokazuje przeglądarce jakie treści ma wyświetlać i jakie jest znacznenie tych treści. To tutaj pokazujemy, która treść jest nagłówkiem, która listą, a która paragrafem. Pliki, zawierające kod HTML maja zwykle rozszerzenie .html

HTML składa się ze znaczników / tagów / elementów

<znacznik atrybut="wartość">treść</znacznik>

Uczymy się pisać kod semantyczny. Semantyczny czyli mający znaczenie. Strony, które są napisane poprawnie pod względem semantycznym są lepiej indeksowane przez przeglądarki. Takie strony są także lepsze dla czytników, ktrórych używają osoby niewidome

Poznane znaczniki:

  • main - główna część dokumentu
  • section - sekcje dokumentu np. tak jak sekcja sportowa w gazecie
  • article - samodzielne treści np. tak jak pojedynczy artykuł w sekcji sportowej
  • header - nagłówek dokumentu / sekcji / artykułu
  • footer - stopka dokumentu / sekcji / artykułu
  • h1 - h6 - tekstowe nagłówki tytułowe dokumentu / sekcji
  • p - paragraf / akapit
